---
title: Buffer time settings
description: Control how much breathing room Paceday adds before and after your meetings, so you always have time to prepare and decompress.
---

Buffer time is the cushion Paceday adds around your meetings — a few minutes before to prepare, and a few minutes after to decompress or finish notes. You can configure exactly how much buffer you want, and whether back-to-back meetings should skip the buffer between them.

## Where to find buffer time settings

Go to **Settings > Buffer time**. All controls are on one page.

## Enable or disable buffer time

At the top of the page, there's a toggle for **Buffer time**. Turn it on to have Paceday automatically block time around meetings. Turn it off to stop adding buffers altogether — your existing buffer blocks on the calendar will be removed at the next sync.

## Set minutes before meetings

Use the **Before meetings** field to set how many minutes Paceday should block before each meeting. Common choices are 5 or 10 minutes. This time is marked as busy on your calendar so others can't schedule over it.

## Set minutes after meetings

Use the **After meetings** field to set how many minutes Paceday should block after each meeting. Many people use this for notes, follow-ups, or just a moment to reset before the next thing.

## Skip buffer for back-to-back meetings

If you have two meetings in a row — say, one ending at 10:00 AM and another starting at 10:00 AM — adding buffer between them would require pushing one of the meetings, which isn't always possible.

The **Skip buffer between back-to-back meetings** toggle handles this gracefully. When it's on, Paceday won't try to add buffer in the gap between consecutive meetings. Buffer is still added before the first meeting and after the last one in the sequence.

Turn this off if you'd prefer Paceday to always enforce the full buffer, even when it means flagging scheduling conflicts.

## When changes take effect

Buffer time settings apply immediately. As soon as you save, Paceday updates your calendar — adding or removing buffer blocks within a few seconds. You don't need to restart or re-sync anything.
