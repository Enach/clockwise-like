## Context
The app auto-schedules "Focus Time" blocks on the user's Google Calendar. Focus blocks are calendar events
marked as "Free" (so they don't block meeting invites), labeled with the user's configured label and color.
The engine runs on a cron schedule (default: weekdays at 7am) and can also be triggered manually.

## Task: Implement the Focus Time engine

### engine/focus_time.go

Implement `FocusTimeEngine` struct with method:
`func (e *FocusTimeEngine) Run(ctx context.Context, targetWeek time.Time) (*FocusRunResult, error)`

**Algorithm (detailed):**
1. Load settings from DB
2. For each weekday of the target week (Mon–Fri, or target day if running for today):
   a. Fetch all calendar events for that day (already existing focus blocks + meetings)
   b. Identify existing app-owned focus blocks (from focus_blocks table) — skip days that already meet the daily target
   c. Build a list of "busy" intervals: all non-focus meetings within working hours
   d. If protect_lunch: add lunch window as a busy interval
   e. Compute free slots = working hours MINUS busy intervals
   f. Filter free slots: keep only slots ≥ focus_min_block_minutes
   g. Score remaining free slots:
      - +30 pts if slot starts before 12:00 (morning preferred)
      - +20 pts if slot is between 14:00–17:00 (good afternoon)
      - +10 pts per 30 minutes of duration (longer = better)
      - -10 pts if slot is Monday morning (often full of meetings)
   h. Sort slots by score descending
   i. Greedily pick slots until daily target is met:
      - Clip slot to max_block_minutes if too long
      - Add buffer_before and buffer_after around the block
3. For each chosen slot, create a Google Calendar event:
   - Title: settings.FocusLabel
   - Color: settings.FocusColor
   - Status: "free" (transparency: "transparent")
   - Description: "Scheduled by Clockwise-like"
   - Visibility: "public"
4. Save created event IDs to focus_blocks table
5. Write to audit_log: action="focus_created", details=JSON with date, count, total_minutes

**FocusRunResult struct:**
```go
type FocusRunResult struct {
  WeekStart     time.Time
  CreatedBlocks []FocusBlock  // newly created
  SkippedDays   []string      // dates skipped (already had enough focus time)
  TotalMinutes  int
  Errors        []string      // per-day errors (don't abort whole run)
}
```

### engine/focus_time_cleaner.go
`func (e *FocusTimeEngine) ClearWeek(ctx, targetWeek time.Time) error`
- Load focus_blocks for the week from DB
- Delete corresponding Google Calendar events
- Remove from focus_blocks table
- Write audit_log: action="focus_cleared"

### api/handlers_focus.go
- `POST /api/focus/run` — body: `{ "week": "2026-04-27" }` (optional, defaults to current week)
    - Calls engine.Run(), returns FocusRunResult as JSON
- `GET /api/focus/blocks?week=2026-04-27` — return focus blocks for the week from DB
- `DELETE /api/focus/blocks?week=2026-04-27` — calls ClearWeek, returns count deleted
