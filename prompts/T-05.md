## Context
Meeting compression = move meetings to consolidate them into blocks, creating larger contiguous free slots
(ideal for Focus Time). The engine never auto-applies moves — it only proposes them. The user reviews and
applies proposals via the UI.

Moveable meetings = meetings where:
- The user is the organizer, AND
- All attendees are available at the proposed new time (checked via freebusy API)

Non-moveable meetings = external meetings (attendees outside org), all-day events, events with RSVP declined.

## Task: Implement the Meeting Compression engine

### engine/compression.go

**Structs:**
```go
type MoveProposal struct {
  EventID       string
  EventTitle    string
  CurrentStart  time.Time
  CurrentEnd    time.Time
  ProposedStart time.Time
  ProposedEnd   time.Time
  Reason        string  // e.g., "Moves to create 2h focus block in the afternoon"
  FocusGainMinutes int  // how many focus minutes this move creates
}

type CompressionResult struct {
  Date      string
  Proposals []MoveProposal
  TotalFocusGainMinutes int
}
```

**Algorithm:**
`func (e *CompressionEngine) SuggestForDay(ctx, date time.Time) (*CompressionResult, error)`
1. Fetch all events for the day
2. Filter to moveable meetings (user is organizer, not all-day)
3. Compute current free slots for the day (within working hours, minus lunch)
4. Compute largest_current_free_slot
5. For each permutation of moving meetings to different positions in the day:
    - Only consider positions that keep meetings back-to-back (morning block or afternoon block)
    - Check attendee availability via GetFreeBusy for the proposed time
    - Compute resulting free slots
    - Score = largest resulting contiguous free slot
6. Return top proposal (or top 3 if multiple significantly different options)
7. Only return proposals where FocusGainMinutes > 30 (not worth it for smaller gains)

Note: For performance, don't try all permutations for days with many meetings.
Use a greedy approach: for each meeting, try moving it to be adjacent to the nearest other meeting.
