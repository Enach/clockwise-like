# T-24 тАФ Backend: Auto-associate users by custom email domain (Organizations)

## Goal
On every OAuth sign-in, detect the user's email domain and automatically place them in an organization with other users from the same domain. Generic provider domains (gmail.com, outlook.com, etc.) are excluded.

## Logic (runs after T-22 user upsert)
1. Extract domain from email (`nicolas@gorgias.com` тЖТ `gorgias.com`)
2. If domain is in the generic blocklist тЖТ skip, `user.org_id` stays NULL
3. `UpsertOrg(domain, deriveOrgName(domain))` тАФ creates org on first user, reuses on subsequent
4. `SetUserOrg(userID, org.ID)` тАФ links user to org
5. `GET /api/auth/me` now returns `org: { id, name, domain }` when user has an org

## Org name derivation
`"gorgias.com"` тЖТ `"Gorgias"`, `"acme-corp.io"` тЖТ `"Acme Corp"` (hyphen тЖТ space, each word capitalized)
