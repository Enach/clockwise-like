You are a senior DevOps and Go engineer. Migrate the Clockwise-like backend database from SQLite to PostgreSQL 16.

## Context

- Current stack uses `modernc.org/sqlite` with an in-process SQLite file at `/data/clockwise.db`
- Target: PostgreSQL 16 running as a sidecar container in Docker Compose
- Every subsequent task (T-11 through T-16, T-18) assumes PostgreSQL — this task MUST run first

## Step 1 — docker-compose.yml

Add a `postgres:16-alpine` service with a healthcheck. Update the `backend` service:
- Remove `DB_PATH` env var
- Add `DATABASE_URL=postgres://clockwise:${POSTGRES_PASSWORD:-clockwise}@postgres:5432/clockwise?sslmode=disable`
- Add `depends_on: postgres: condition: service_healthy`
- Replace `sqlite_data` volume with `postgres_data`

## Step 2 — Go dependencies

In `backend/`:
```bash
go get github.com/jackc/pgx/v5
go get github.com/golang-migrate/migrate/v4
go mod edit -droprequire modernc.org/sqlite
go mod tidy
```

## Step 3 — Embedded SQL migrations

Create `backend/storage/migrations/001_initial.up.sql` with all tables in PostgreSQL syntax:
- `SERIAL PRIMARY KEY` instead of `INTEGER PRIMARY KEY AUTOINCREMENT`
- `BOOLEAN` instead of `INTEGER` for boolean columns
- `TIMESTAMPTZ` instead of `DATETIME`
- `NOW()` instead of `CURRENT_TIMESTAMP`

Create `backend/storage/migrations/001_initial.down.sql` with `DROP TABLE IF EXISTS` for all tables.

## Step 4 — Rewrite storage/db.go

Use `//go:embed migrations/*.sql`, pgx stdlib driver (`_ "github.com/jackc/pgx/v5/stdlib"`), and golang-migrate with iofs source:

```go
//go:embed migrations/*.sql
var migrationFiles embed.FS

func Open(dsn string) (*sql.DB, error) { ... }
func runMigrations(db *sql.DB) error { ... }
```

## Step 5 — Port storage/*.go to PostgreSQL syntax

- Replace `?` placeholders with `$1`, `$2`, etc.
- Replace `CURRENT_TIMESTAMP` with `NOW()`
- Replace `INSERT OR IGNORE` with `INSERT ... ON CONFLICT DO NOTHING`
- Replace SQLite-specific functions (`julianday`, `last_insert_rowid`) with PostgreSQL equivalents
- Scan `bool` and `time.Time` directly (no intermediate string conversion)
- Remove `boolToInt()` helper

## Step 6 — Update tests to use testcontainers-go

Replace `:memory:` SQLite in test helpers with `postgres.RunContainer` from `testcontainers-go`.

## Step 7 — Update main.go

Read `DATABASE_URL` from environment instead of `DB_PATH`.

## Step 8 — Makefile

Add `db-shell` target:
```makefile
db-shell:
	docker compose exec postgres psql -U clockwise -d clockwise
```
