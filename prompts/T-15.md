# T-15 — Meeting editor (location, rooms, participants)

## Goal
Expose PATCH/DELETE for individual calendar events and two helper endpoints for room discovery and attendee auto-complete.

## New file: `api/handlers_events.go`

### PATCH /api/events/:id
Partial update — only provided fields are changed:
```json
{ "title": "...", "description": "...", "location": "...", "start": "RFC3339", "end": "RFC3339", "attendees": ["a@b.com"] }
```
Flow: `GetEvent` → mutate → `UpdateEvent` → return updated event JSON.

### DELETE /api/events/:id
Calls `client.DeleteEvent` and returns 204.

### GET /api/rooms?q=
Calls `CalendarClient.ListRooms` which queries `CalendarList.List()` and filters
entries whose ID contains `resource.calendar.google.com`. Optional `q` prefix filter.
Returns `[{id, name}]`.

### GET /api/attendees/suggest?q=
Calls `CalendarClient.SuggestAttendees` which scans events from the last 30 days,
harvests unique attendee emails, and filters by optional `q` prefix.
Returns up to 20 `[{email, name}]`.

## New file: `calendar/rooms_attendees.go`
- `Room` and `AttendeeSuggestion` structs
- `(c *CalendarClient) ListRooms(ctx, query) ([]Room, error)`
- `(c *CalendarClient) SuggestAttendees(ctx, query, lookback) ([]AttendeeSuggestion, error)`
