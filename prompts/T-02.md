## Task: Implement Google OAuth2 + Calendar integration

### Go dependencies to add (go get)
- golang.org/x/oauth2
- golang.org/x/oauth2/google
- google.golang.org/api/calendar/v3
- google.golang.org/api/option

### Files to create/modify

1. **auth/google_oauth.go**
   - `NewGoogleOAuthConfig(clientID, clientSecret, redirectURL string) *oauth2.Config`
     - Scopes: calendar.ReadWrite, userinfo.email
   - `GetAuthURL(config *oauth2.Config, state string) string`
   - `ExchangeCode(ctx, config, code) (*oauth2.Token, error)`
   - `TokenFromDB(db *storage.DB) (*oauth2.Token, error)` — load stored token
   - `SaveToken(db *storage.DB, token *oauth2.Token) error` — upsert into oauth_tokens
   - `TokenSource(ctx, config, token) oauth2.TokenSource` — auto-refreshing token source

2. **auth/token_store.go**
   - CRUD for oauth_tokens table: Save, Load, Delete, IsConnected

3. **calendar/client.go**
   - `NewClient(ctx, tokenSource) (*CalendarClient, error)` — wraps Google Calendar v3 service
   - Struct `CalendarClient` with the service and calendar ID

4. **calendar/events.go**
   - `ListEvents(ctx, calendarID, timeMin, timeMax time.Time) ([]*calendar.Event, error)`
   - `CreateEvent(ctx, calendarID string, event *calendar.Event) (*calendar.Event, error)`
   - `UpdateEvent(ctx, calendarID, eventID string, event *calendar.Event) (*calendar.Event, error)`
   - `DeleteEvent(ctx, calendarID, eventID string) error`
   - `GetEvent(ctx, calendarID, eventID string) (*calendar.Event, error)`

5. **calendar/freebusy.go**
   - `GetFreeBusy(ctx, emails []string, timeMin, timeMax time.Time) (map[string][]TimeSlot, error)`
   - `TimeSlot` struct: `Start, End time.Time`
   - Returns a map from email → list of busy slots

6. **api/handlers_auth.go**
   - `GET /api/auth/google` — generate state (random 16-byte hex), store in a short-lived cookie, redirect to Google OAuth URL
   - `GET /api/auth/callback` — validate state cookie, exchange code, save token to DB, redirect to `/?connected=true`
   - `GET /api/auth/status` — return `{ "connected": bool, "email": "user@gmail.com" }` (fetch email from Google userinfo if connected)
   - `DELETE /api/auth/disconnect` — delete token from DB, return 200

7. **api/handlers_calendar.go**
   - `GET /api/calendar/events?start=&end=` — list events, return JSON array
   - `GET /api/calendar/freebusy?start=&end=&attendees=a@b.com,c@d.com` — return free/busy

### Environment variables used
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- GOOGLE_REDIRECT_URL (e.g. http://localhost/api/auth/callback)
- APP_SECRET (for signing state cookie / session)
