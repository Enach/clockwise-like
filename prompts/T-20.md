# T-20 — MCP Server (Claude Desktop / Cursor / Zed integration)

## Goal
Expose Clockwise-like as an MCP (Model Context Protocol) server so that AI coding assistants and Claude Desktop can call calendar tools directly.

## Architecture
```
MCP client (Claude Desktop / Cursor / Zed)
        ↕ MCP protocol (stdio or SSE)
  mcp-server  :3001  (Go binary, ./mcp)
        ↕ HTTP
    backend   :8080
```

## Transports
Selected via `MCP_TRANSPORT` env var:
- `sse` — HTTP SSE endpoint, suitable for web clients and docker-compose deployment
- anything else (default) — stdio, suitable for subprocess mode (Claude Desktop, Cursor config)

## Library
`github.com/mark3labs/mcp-go v0.17.0`

## Files
- `mcp/go.mod` — standalone Go module
- `mcp/main.go` — stdio/SSE dispatch
- `mcp/client.go` — HTTP adapter for backend
- `mcp/tools.go` — all tool registrations
- `mcp/Dockerfile` — multi-stage Go build

## Docker Compose
`mcp-server` service added, port 3001 exposed.

## Claude Desktop config (stdio mode)
```json
{
  "mcpServers": {
    "clockwise": {
      "command": "/path/to/mcp-server",
      "env": { "BACKEND_URL": "http://localhost:8080" }
    }
  }
}
```

## Env vars
| Var | Purpose |
|---|---|
| `BACKEND_URL` | Base URL of the backend (default: http://localhost:8080) |
| `MCP_TRANSPORT` | `sse` or empty for stdio |
| `MCP_PORT` | SSE listen port (default: 3001) |
| `MCP_AUTH_TOKEN` | Optional bearer token to protect the SSE endpoint |
