# T-13 — Multi-provider work calendar (Outlook + WebCal)

## Goal
Add Microsoft Outlook and WebCal (iCal feed) as alternative calendar providers alongside Google Calendar. The active provider is stored in `settings.calendar_provider` and controls which backend is used for read/write operations.

## DB migration (`003_calendar_providers`)
```sql
ALTER TABLE settings ADD COLUMN calendar_provider TEXT NOT NULL DEFAULT 'google';
ALTER TABLE settings ADD COLUMN microsoft_tokens TEXT;
ALTER TABLE settings ADD COLUMN webcal_url TEXT NOT NULL DEFAULT '';
ALTER TABLE settings ADD COLUMN calendar_email TEXT NOT NULL DEFAULT '';
```

## Provider interface (`calendar/provider.go`)
```go
type GenericEvent struct { ID, Title string; Start, End time.Time }
type Provider interface {
    ListEvents(ctx, start, end) ([]GenericEvent, error)
    GetFreeBusy(ctx, emails, start, end) (map[string][]TimeSlot, error)
    CreateEvent(ctx, e GenericEvent) (string, error)
}
var ErrReadOnly = errors.New("calendar: provider is read-only")
```

## Outlook client (`calendar/outlook_client.go`)
Calls Microsoft Graph API using `oauth2.TokenSource`.
- `ListEvents` → GET `/me/calendarView?startDateTime=...&endDateTime=...`
- `GetFreeBusy` → POST `/me/calendar/getSchedule`
- `CreateEvent` → POST `/me/events`

## WebCal client (`calendar/webcal_client.go`)
Fetches and parses read-only iCal feeds using `github.com/emersion/go-ical`.
- Responses cached 15 minutes in memory.
- `CreateEvent` returns `ErrReadOnly`.

## Provider factory (`calendar/client_factory.go`)
`NewProvider(ctx, *storage.Settings, oauth2.TokenSource) (Provider, error)` dispatches on `settings.CalendarProvider`.
The existing `*CalendarClient` is wrapped in a `googleProvider` adapter.

## Microsoft auth (`auth/microsoft_oauth.go`)
```go
type MicrosoftConfig struct { ClientID, ClientSecret, RedirectURL string }
func GetMicrosoftAuthURL(cfg *MicrosoftConfig, state string) string
func ExchangeMicrosoftCode(ctx, cfg *MicrosoftConfig, code string) (*oauth2.Token, error)
func SaveMicrosoftToken(db, token) error  // stores JSON in settings.microsoft_tokens, sets provider='outlook'
func LoadMicrosoftToken(db) (*oauth2.Token, error)
```

## New API routes
```
GET  /api/auth/microsoft            — redirect to Microsoft consent page
GET  /api/auth/microsoft/callback   — exchange code, save token, redirect /?connected=true&provider=outlook
GET  /api/auth/status               — now returns { connected, provider, email }
```
